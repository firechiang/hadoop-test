#### 使用说明：
```bash
1，保存proto文件一般会自动编译成JAVA文件，如果没有自动编译，刷新项目就会自动编译
2，安装设置好设置项以后需要重启Eclipse，设置项才会生效
3，每次编译完成后会报一个Xtext的错误，这个不影响使用，可以不管它
```
